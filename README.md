# youapp-be-test
Assessment Techincal Test for YouApp Backend

## Installation

1. Clone the project

```bash
  git clone https://github.com/widz11/youapp-be-test.git
```

2. Go to the project directory

```bash
  cd youapp-be-test
```

3. Create database and queue for "chat"

4. Setup env

```bash
  cp enc-example .env
```

5. Install dependencies

```bash
  npm install
```

6. Start the server

```bash
  npm run start
```

7. Go to service

```bash
  http://localhost:3000
```

8. Run test

```bash
  npm run test
```
